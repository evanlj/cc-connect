# instance-mutilbot4

角色：**行走**（测试与缺陷专家）

## 需要你手动填写

1. `config.toml` / `codez.toml` / `rigthcode.toml` 中的：
   - `projects.agent.providers[*].api_key`
2. 飞书配置（你已说明手动填写）：
   - `projects.platforms.options.app_id`
   - `projects.platforms.options.app_secret`

## 启动方式

- `start-instance-mutilbot4-codez.bat`
- `start-instance-mutilbot4-rightcode.bat`

> 说明：默认 provider 分别对应 `codez` 与 `rightcode`。
